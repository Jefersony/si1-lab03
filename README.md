# Find-Bandmate
Um site para procurar companheiros de banda.

Este projeto foi desenvolvido com o framework Play 1.3.0.


OBS:
		
Nos testes ha operacoes de delecao completa do BD, por isso, no modo play test, 
note que os objetos criados ao iniciar o site serao deletados. So ha garantia de 
nao haver NullPointer Exception se os itens do site forem acessados no modo play run.

Na página de busca, pesquise pela tag Guitarra e os anúncios finalizados aparecerao.
